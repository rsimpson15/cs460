The Site

We have to create a multi-page website that incorperates the following elements: Bootstrap, a List (Ordered, Unordered and Description List), a Table, CSS files, a Nav Bar, and consistent style. We were also not allowed to use major wigwams so I opted for the permitted choice of Brackets.

The site I designed is called Art of Earth and is made for a fictional small company that creates crystal trees. The pages include an Index, an About, the Shamanic Tools, the Custom Make and a Contact page. There is also three different CSSs (that are not Boostrap) - one for general cohesion formatting, another for certain page specfic elements that weren't needed across the site and another for print readiness. There is also some Javascript for an image gallery on one of the pages.
