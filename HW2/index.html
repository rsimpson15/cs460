Cupertino & Data Tables

Being somewhat familiar with the libraries made for Javascript, I opted to create a database for my project. This would not only react to different feilds and sort them in ascending or descending order, but it also has a search function to pluck out a specific person from the databse if they are listed.

Cupertino is a CSS Framework designed by multiple people that was created to assist with organizing tables in a pleasing manner for this purpose. It also places the images that make the interface look more elegant like directional arrows.

Data Tables is the same thing except it is a Framework for Javascript that programs the user interactivity. It was designed by Allan Jardine, copyright 2008 - 2012.

The elemets are implimented by a simple but elegant bit of personal code.

	   });
